# Error Responses

## Validation Error (400)

```json
{
    "error": {
        "issues": [
            {
                "code": "too_small",
                "minimum": 1,
                "type": "number",
                "inclusive": true,
                "exact": false,
                "message": "Number must be greater than or equal to 1",
                "path": ["page"]
            }
        ],
        "name": "ZodError"
    }
}
```

## Authentication Error (401)

```json
{
    "error": "X-API-Key header is required"
}
```

