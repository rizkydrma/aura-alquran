# Rate Limiting

This API limits the number of requests to prevent abuse and maintain system performance.

- **Limit**: Maximum 100 requests per window per API key
- **Time Window**: 60 seconds (1 minute)

## Rate Limit Headers

Each response will include the following headers to help you monitor usage:

- `X-RateLimit-Limit`: Maximum number of requests allowed per window (100)
- `X-RateLimit-Remaining`: Number of requests remaining in the current window

## When Limit is Exceeded

If you exceed the request limit, the system will return the following response:

```json
{
    "error": "Rate limit exceeded"
}
```

**Status Code**: `429`

Additional headers when limit is exceeded:

- `Retry-After`: Wait time in seconds before the next request is allowed
