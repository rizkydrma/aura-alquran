# Ninetynine X Al-Qur'an REST API Documentation

Backend API for accessing Al-Qur'an data using Hono.js, Bun, TypeScript, PostgreSQL, and Drizzle ORM.

## Base URL

```
http://localhost:3000/api/v1
```

## Overview

This API provides full access to Al-Qur'an data complete with translations, transliterations, and mushaf structure. All endpoints require authentication using the `X-API-Key` header.

Key features:

- Access to all 114 Surahs
- Ayat search with full-text search
- Pagination for handling large datasets
- Rate limiting to prevent abuse
- Structured responses with Zod validation
